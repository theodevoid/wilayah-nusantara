# Get District by Area Code

Fetches a district based on its area code.

### Endpoint

```md filename="Base URL"
https://api.wilayah-nusantara.id

or, if running on local

http://localhost:1945
```

```md filename="Method and Endpoint"
GET /kabupaten/{code}
```

### Route Params

##### `code` : string (required)

Area code of a district.

### Responses

#### 200 (success)

```md filename="Example Request"
GET https://api.wilayah-nusantara.id/kecamatan/3674020
```

```json filename="Example Response"
{
  "code": "3674020",
  "district": "3674",
  "district": "SERPONG"
}
```

#### 404 (not found)

```json filename="Example Response"
{
  "statusCode": 404,
  "message": "district not found"
}
```

#### 500 (server error)

```json filename="Example Response"
{
  "statusCode": 500,
  "message": "Internal server error"
}
```
